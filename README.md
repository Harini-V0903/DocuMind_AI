## DocuMind_AI

## Overview

StudyMate AI is a lightweight Retrieval-Augmented Generation (RAG) based application that allows users to upload PDF documents and ask natural language questions. The system extracts content from documents, retrieves relevant context, and generates intelligent answers using Groq LLM (LLaMA 3).

## Features

  Upload and read PDF documents
  
  Extract and process text automatically
  
  Split text into meaningful chunks
  
  Retrieve relevant context using keyword-based search
  
  Generate AI-powered answers using Groq LLM
  
  Simple and interactive Streamlit UI

## Tech Stack

  Python
  
  Streamlit
  
  PyPDF2
  
  Groq API (LLaMA 3)
  
  dotenv (for secure API key management)

## Learning Outcome

This project demonstrates:
Basics of RAG pipelines

Document processing techniques

LLM integration using APIs

Building interactive AI web apps using Streamlit
